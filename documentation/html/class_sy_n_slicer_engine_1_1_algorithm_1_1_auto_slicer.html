Determine all default intermediate planes between plane_up and plane_below.

Default intermediate planes may not meet the requirement of layer thickness in between m_min_layer_thickness and m_max_layer_thickness.
All default intermediate planes will adjust in calculateIntermediatePlanes() so that all of them can meet the requirement.

bool AutoSlicer::determineNextSlicingPlane ( SO::PolygonCollection & current_starting_contours,
SO::PolygonCollection & next_starting_contours )
protectedvirtual

Offset slicing plane and search for .

Part 1: Offset the current slicing plane by target_layer_thickness continuously until support structure is needed.
Part 2: Base on the starting slicing plane and the last slicing plane, determine all the intermediate planes that can minimize support structure.

Parameters
current_starting_contoursThe contours used to start offsetting this time.
next_starting_contoursThe contours used to start offsetting next time.
