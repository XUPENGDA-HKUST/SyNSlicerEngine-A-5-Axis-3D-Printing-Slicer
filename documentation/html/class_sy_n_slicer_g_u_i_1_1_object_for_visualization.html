Detailed Description

This class is used to store a object for visualizing in the renderer.

For every item you want to display, add this one into item's private member Then create the item's polydata, use setPolyData() to store the polydata and then call addToRenderer() to show the item on screen Delete this class will automatically remove the item from the screen.
