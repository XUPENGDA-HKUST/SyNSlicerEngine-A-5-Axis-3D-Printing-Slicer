Detailed Description

template<class T>
class SyNSlicerEngine::Object::Partition< T >

This class is used describe the 3D model after partitioning.

The difference between this and the superclass Model is that it stores members.
(1) A contour obtain during parititioning used to calculate printing sequence.
(2) Partition that should be printed before this partition does.
(3) All printing layers obtained after slicing this partition.
